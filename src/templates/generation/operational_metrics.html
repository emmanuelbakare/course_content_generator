Authorization monitoring

This application does not persist access-attempt events. Maintain automated ownership-isolation tests and alert on unexpected 403/404 authorization responses in structured web logs. The target is zero successful cross-owner access attempts.

Calculation notes

  • Generation rates include terminal jobs created in the window; provider-configuration and invalid-target failures are excluded. Non-terminal jobs are excluded.
  • Export rates include all terminal export jobs created in the window; cancelled exports count as unsuccessful.
  • Approval-or-edit counts successful generated curricula that are currently approved or have a later immutable revision.
  • Approval time is a persistent-record proxy: course creation to the first approved/superseded curriculum version's creation time. The application does not yet record the exact approval-click time.
